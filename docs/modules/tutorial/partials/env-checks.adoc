which apptainer || true

which sbatch || true
which squeue || true
----

[.note]
====
`kub-cli` requires one of `apptainer` or `docker` to be available on `PATH` when using local runs.
If you are using containerized simulation through the `docker` runtime, install Docker Engine.
If you are using `apptainer`, install Apptainer on the host.
====
